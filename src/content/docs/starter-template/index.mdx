The starter template is the base application that modules install into. It provides:

- **App shell** - Sidebar, header, layout components
- **Authentication** - Neon Auth integration
- **Registries** - Extension points for modules
- **Base features** - User settings, preferences, dashboard

## Tech Stack

| Layer | Technology |
|-------|------------|
| Build | Vite + SWC |
| Frontend | React 19 + TypeScript |
| Styling | Tailwind CSS v4 |
| UI Components | shadcn/ui |
| Icons | Lucide (strokeWidth: 1) |
| Backend | Express.js + TypeScript |
| Database | Neon PostgreSQL + Prisma |
| Auth | Neon Auth |

## Base Features

### auth

Authentication pages and session management using Neon Auth.

### dashboard

App shell components: sidebar, header, layout wrappers.

### user

User settings, preferences, and profile management.

## Registries

The starter template includes these registries ready for modules to use:

### Frontend Registries

| Registry | Purpose |
|----------|---------|
| navigation | Sidebar and menu items |
| dashboard | Dashboard widgets |
| preferences | Settings sections |
| integrations | Third-party connections |
| settings-pages | Settings page components |
| personal-info-fields | Profile form fields |
| filters | Data transformation hooks |

### Backend Registries

| Registry | Purpose |
|----------|---------|
| routes | API route handlers |
| services | Cross-feature services |
| filters | Request/response transformation |

## Quick Start

### 1. Clone the Template

```bash
git clone https://github.com/ui-git-kim/unique-react-fullstack-starter.git my-app
cd my-app
rm -rf .git
git init
```

### 2. Install Dependencies

```bash
npm run install:all
```

### 3. Configure Environment

```bash
cp backend/.env.example backend/.env
# Edit with your database credentials
```

### 4. Set Up Database

```bash
cd backend
npx prisma db push
```

### 5. Run the App

```bash
npm run dev
```

### 6. Add a Module

```bash
npx billing-module init
```

Follow the prompts. The module installs itself, registers with the app, and you're done.
