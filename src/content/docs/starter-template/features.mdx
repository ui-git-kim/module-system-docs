## User Profile

**Location:** `frontend/src/features/user/`

```typescript
import {
  useUserProfile,    // Profile data and role checks
  UserAvatar,        // Avatar component
} from '@/features/user';

// Usage
const { profile, isAdmin, isLoading } = useUserProfile();
```

### UserAvatar

```typescript
<UserAvatar />                    // Auto-loads current user
<UserAvatar size="lg" />          // sm, md, lg, xl
<UserAvatar imageUrl={url} />     // Override image
```

## Backend Services

### User Service

```typescript
import { userService } from './features/user/user.service.js';

await userService.getOrCreateProfile(userId, email);
await userService.updateProfile(userId, { displayName: 'Name' });
await userService.isAdmin(userId);
```

### Auth Middleware

```typescript
import { requireAuth, requireAdmin, requireRole } from '../middleware/auth.middleware.js';

router.use('/api', requireAuth, routes);
router.use('/admin', requireAuth, requireAdmin, adminRoutes);
```

### Error Classes

```typescript
import { NotFoundError, BadRequestError, UnauthorizedError } from '../../lib/errors.js';

if (!item) throw new NotFoundError('Item not found');
```

## Extending Features

Use registries to add functionality:

```typescript
// Add navigation
navigationRegistry.registerMain({
  id: 'my-feature',
  title: 'My Feature',
  url: '/my-feature',
  icon: MyIcon,
  order: 50,
});

// Add dashboard widget
dashboardRegistry.register({
  id: 'my-widget',
  title: 'My Widget',
  component: MyWidget,
  size: '2x1',
});

// Add settings page
settingsPagesRegistry.register({
  id: 'my-settings',
  title: 'My Settings',
  component: MySettingsContent,
});
```
