## Frontend Registries

### navigationRegistry

```typescript
navigationRegistry.registerMain({
  id: string;
  title: string;
  url: string;
  icon?: LucideIcon;
  order?: number;
  items?: { title: string; url: string }[];
});

navigationRegistry.registerUserMenu({ ... });
navigationRegistry.getMain(): NavigationItem[];
navigationRegistry.getUserMenu(): NavigationItem[];
```

**Hook:** `useNavigationRegistry()`

---

### dashboardRegistry

```typescript
dashboardRegistry.register({
  id: string;
  title: string;
  component: ComponentType<DashboardWidgetProps>;
  size: '1x1' | '2x1' | '1x2' | '2x2' | 'full';
  order?: number;
  category?: string;
  canHide?: boolean;
});
```

**Hook:** `useDashboardRegistry()`

---

### settingsPagesRegistry

```typescript
settingsPagesRegistry.register({
  id: string;
  title: string;
  description?: string;
  icon?: LucideIcon;
  component: ComponentType;
  order?: number;
});
```

**Hook:** `useSettingsPagesRegistry()`

---

### headerToolbarRegistry

```typescript
headerToolbarRegistry.register({
  id: string;
  type: 'action' | 'toggle' | 'dropdown' | 'custom';
  position: 'left' | 'right';
  order?: number;
  icon?: LucideIcon;
  label?: string;
  onClick?: () => void;
  badge?: number;
  component?: ComponentType;
});

headerToolbarRegistry.update(id, { badge: 5 });
```

**Hook:** `useHeaderToolbarRegistry()`

---

### filterRegistry

```typescript
filterRegistry.register(hookName, filterId, callback, order);
filterRegistry.apply(hookName, value, context);
filterRegistry.removeFilter(hookName, filterId);
```

---

### actionRegistry

```typescript
actionRegistry.addAction(hookName, actionId, callback, order);
actionRegistry.addActionAsync(hookName, actionId, asyncCallback, order);
actionRegistry.doAction(hookName, context);
actionRegistry.doActionAsync(hookName, context);
```

## Backend Registries

### routeRegistry

```typescript
routeRegistry.register({
  path: string;
  router: Router;
  feature: string;
  versions: string[];
  requiresAuth?: boolean;
});
```

---

### serviceRegistry

```typescript
// Register
declare module '../../config/service.registry.js' {
  interface ServiceTypes {
    'myService': typeof myService;
  }
}
serviceRegistry.register('myService', myService);

// Access
const service = serviceRegistry.get('myService');
```
