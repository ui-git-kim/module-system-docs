## Installing Modules

All modules use the scaffold system:

```bash
npx [module-name] init
```

For example:
```bash
npx structure init
npx billing init
```

After installation:
1. Run `npx prisma generate`
2. Run `npx prisma migrate dev`
3. Follow module-specific post-install steps
