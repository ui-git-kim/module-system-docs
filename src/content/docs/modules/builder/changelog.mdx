## v1.11.11 — February 2026

**Current release.**

- **Feature:** Extensible iframe preview CSS via `builder.canvas.globalCss` filter hook
- Replace `foundry:theme-update` postMessage with `foundry:global-css` (sends full CSS string instead of individual variables)
- Add `buildPreviewCss()` helper for constructing base CSS from project theme variables
- IframeCanvas subscribes to `filterRegistry.subscribe()` for reactive filter updates
- Thread `projectId` through BuilderLayout → PreviewPanel → IframeCanvas
- CSS variables now include `hsl()` wrapper for universal format support (HSL and OKLCH)

## v1.11.10

- **Feature:** Extensible `components.json` snippet via `builder.registry.componentsJson` filter hook
- ProjectRegistryTab uses `filterRegistry.apply()` to allow companion modules to enrich the registry config (e.g. `$schema`, `style`, `tailwind`, `aliases`)

## v1.11.9

- Fix: Breadcrumb cleanup pattern in BuilderPage `useEffect` — properly restores breadcrumbs on unmount

## v1.11.8

- Migrate all backend `console.log`/`console.warn`/`console.error` calls to `createModuleLogger()` structured logging
- Add `builder_config` to `schema.modelNames` in module definition
- Uncomment personal-info registration in module definition
- Remove frontend `console.log` stub handlers (replaced with no-op)
