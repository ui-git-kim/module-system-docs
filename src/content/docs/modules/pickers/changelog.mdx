## v1.2.0 — 2025-02-06

**Initial feature-complete release**

### Picker Components
- **ColorPicker** — Canvas-based HSL picker with solid, gradient, and image fill modes, alpha channel, eyedropper API, preset swatches, and hex/RGB/HSL/OKLCH format support
- **IconPicker** — Full Lucide icon set browser with search, 42 category filters, and tag-based matching
- **DatePicker** — Calendar date selection with month/year navigation, today button, configurable first day of week
- **DateRangePicker** — Dual-calendar range picker with configurable preset ranges and max range limits
- **TimePicker** — Hour/minute/period selection with 12h and 24h format support and configurable minute intervals
- **TimezonePicker** — Searchable timezone dropdown grouped by region with live UTC offset display
- **ImagePicker** — Drag-and-drop image upload with file type validation, size limits, and preview
- **EmojiPicker** — Categorised emoji grid with search, skin tone support, and recently used tracking
- **FontPicker** — Google Fonts browser with live preview, search, and category filtering
- **AvatarPicker** — Composed picker supporting image upload, Lucide icon selection, and initials with colour backgrounds

### Admin & Profiles
- Admin page for creating, editing, and deleting picker profiles
- Profile system with JSON configuration per picker type
- Default profile support (one default at a time)
- Profile config merge chain: defaults -> profile -> prop overrides

### Hooks & Utilities
- `usePickerProfile` — Load and merge profile configuration with caching
- `useColorPicker` — Full colour picker state management (HSL, gradient, eyedropper)
- `useIconPicker` — Icon search, category filtering, and dataset access
- `IconRenderer` — Render a Lucide icon by name string
- Colour utility library with hex/RGB/HSL/OKLCH conversions

### Backend
- REST API for profile CRUD at `/api/v1/pickers/profiles`
- Service registered in `serviceRegistry` for cross-module access
- Routes registered in `routeRegistry` with v1 API versioning
- Self-contained Prisma schema (`pickers_profile` table)

### Infrastructure
- Navigation sidebar registration (Palette icon, order 50)
- Admin page registration (order 50)
- Showcase page at `/pickers` demonstrating all components
- 14 shadcn/ui component dependencies auto-installed
