Cog Ingest is powered by **[cognitive-db](/cognitive-db/)** — the intelligence engine that does the perceiving, entity extraction, connection discovery, and pattern-finding. The engine is **vendored inside this module** (no separate install or auth) and version-locked to each Cog Ingest release.

<Aside type="tip" title="Full engine documentation">
cognitive-db has its own library docs section. Start with the **[Engine Overview](/cognitive-db/)**.
</Aside>
