## Naming Conventions

| Type | Convention | Example |
|------|------------|---------|
| Module folder | lowercase | `analytics` |
| Routes array | `[module]Routes` | `analyticsRoutes` |
| Service object | `[module]Service` | `analyticsService` |
| Main hook | `use[Module]` | `useAnalytics` |
| Pages | `[Module]Page` | `AnalyticsPage` |

## Index File Pattern

### Frontend

```typescript
// index.ts

// Side-effect imports (registrations)
import './navigation.registration';
import './registrations/dashboard.registration';

// Public exports
export { moduleRoutes } from './routes.config';
export { useModule } from './hooks/useModule';
```

### Backend

```typescript
// index.ts
import './route.registration.js';
import './service.registration.js';

export { moduleRoutes } from './[module].routes.js';
export { moduleService } from './[module].service.js';
```

## Order Convention

| Range | Purpose |
